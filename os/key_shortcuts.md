# Keyboard Shortcuts

## Mac

"Command + I" - Show info

"Command + Space" - Spotlight

"Command + Shift + T" - Reopen close Tab.

## Windows

"F5" - Refresh

"F11" - Full screen Mode

## Firefox

"Command + R" - Refresh window.

"Control + Tab" - Switch between Tabs.

"F12" - Developer

"Command + Shift + F" - FullScreen

"Command + B" - Bookmarks
