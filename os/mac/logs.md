## Crash reports

> Gemini gen

Xcode crash logs on macOS can be found in a specific system folder or viewed via the built-in

**Console** application or the **Xcode Organizer**. 

Method 1: Using the Console App

The [Console app](https://support.apple.com/guide/console/reports-cnsl664be99a/mac) offers a user-friendly interface to view and manage all system and user crash reports. 

- Open the **Console** app from `Applications` > `Utilities` or by using Spotlight search.
- In the sidebar under the **Reports** section, select **Crash Reports** (or **User Reports** in older macOS versions).
- Locate the crash logs that start with `Xcode_` followed by a date and time stamp.
- Right-click the desired log file name and select **Reveal in Finder** to access the actual file.
