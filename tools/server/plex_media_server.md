## Commands

Restart service 

```sh
sudo service plexmediaserver restart
```

You can use `start` or `stop`

Check status

```sh
sudo service plexmediaserver status
```

Update plex on Debian

```bash
sudo apt list --upgradable plexmediaserver
```


```bash
sudo apt upgrade plexmediaserver -y
```
